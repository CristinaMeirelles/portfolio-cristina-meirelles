Bem-vindo ao meu portfólio!  
Este espaço reúne projetos desenvolvidos com foco em desenvolvimento front-end, acessibilidade digital e experiência do usuário.

## 🚀 Sobre

Sou uma profissional em transição para a área de tecnologia, com experiência anterior em atendimento e processos, e atualmente focada em desenvolvimento front-end.

Meu objetivo é criar aplicações funcionais, organizadas e acessíveis, que entreguem uma boa experiência para diferentes perfis de usuários.
